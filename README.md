# ✈️ Uçak Emisyon Analizi

**Aircraft Emission Analysis & Carbon Footprint Calculator**

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 🎯 Proje Hakkında

Uçakların karbon emisyonlarını hesaplayan analiz aracı. Farklı uçak tipleri, rota mesafeleri ve atmosferik koşullara göre emisyon hesaplaması yapar.

## 📊 Özellikler

- ✈️ Uçak tipine göre emisyon hesaplama
- 🛣️ Rota bazlı CO₂ analizi
- 🌍 Atmosferik koşullara göre düzeltme
- 📈 CORSIA uyumlu raporlama
